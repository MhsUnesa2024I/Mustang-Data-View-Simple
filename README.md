# Mustang-Data-View-Simple
It's a basic flask project that implemented CRUD and the 4 SQL item, view, store procedure, function and trigger, under the context of Mustang view.

# Instruction Installation
1. Extract every file.
2. Import the SQL file into XAMPP
3. Run the app.py and ctrl + click the result
4. Enjoy

# Technology Used
- Flask, backend component handling server side logic
- XAMPP : ~ Table, containing the data
          ~ View, to see the table
          ~ Procedure, to add/edit table
          ~ Function, to calculate inside MySQL
          ~ Trigger, to record any delted data
- JavaScript, To handle some table control
